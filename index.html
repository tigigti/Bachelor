    Create your Roadmap

    You can close the Notepad now and start structuring your goals with the Roadmap Editor. You can reopen the Notes on the previous Page as a reference.

    To add a goal, type in the input field where it says "New Goal". If you click on a goal, you can add metadata like a description or date. Click "Save" after configuring it. Connect the Goals by using the drag handler above the Nodes in the Graph.

    Connect the goals that build upon each other. If "graduating" is your meso goal and "getting a job" is a macro goal, connect "graduating" to "getting a job", because the one is necessary for the other (just an example, see which connections make sense to you).

    There are also smaller tasks for each goal you can configure. The "Toggle todo-view" displays every goal and every task in a list view.

    Please evaluate the Application on the next Pages after you have used it to create your hierarchical goal structure.
